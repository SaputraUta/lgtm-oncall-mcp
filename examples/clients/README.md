# Client integration guides

Pick your client:

- [Claude Code (CLI)](./claude-code.md) — interactive terminal copilot
- [Claude Desktop (GUI)](./claude-desktop.md) — interactive desktop app with image rendering
- [Hermes Agent (Nous Research)](./hermes.md) — autonomous on-call agent driven by webhooks

The MCP server itself doesn't care which client you use — it's the same 17 tools either way. These guides only cover the registration syntax and a few client-specific gotchas.

For background on the server and its configuration, see the [main README](../../README.md).
